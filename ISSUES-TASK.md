  - Acceptance evidence:
    - `EmbeddingGateway` depends on an `EmbeddingProvider` protocol; `NvidiaEmbeddingProvider` is the NVIDIA HTTP adapter.

    - Metrics track request count, latest RPM usage, batch sizes, latency, retry count, and failures.
